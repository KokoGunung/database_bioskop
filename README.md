# Cara Run
1. **Import file sql [database_bioskop.sql](/database_bioskop.sql)**
    Download lalu import file [tersebut](/database_bioskop.sql) ke mysql

2. **Masuk ke folder xampp/httdocs**
    Buka folder penyimpanan xampp, cari httdocs

3. **Clone Repositori**
    Clone repositori ini ke xampp/httdocs
    ```bash
    git clone https://github.com/KokoGunung/database_bioskop.git
    ```
4. **Masuk ke localhost**
    Buka browser tempel link ini 
        ```
        localhost/database_bioskop/
        ```
